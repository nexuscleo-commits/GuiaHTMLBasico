Estrutura Básica do HTML

A estrutura básica de um documento HTML inclui as seguintes tags principais:

  • <!DOCTYPE html>: Declara o tipo de documento.
  • <html>: Raiz do documento.
  • <head>: Contém metadados e links para CSS/JS.
  • <title>: O título da página na aba do navegador.
  • <body>: Contém o conteúdo visível da página.
                
                    <!DOCTYPE html>
                    <html lang="pt-br">
                        <head>
                            <meta charset="UTF-8">
                            <title>Título da Página</title>
                        </head>
                        <body>
                            <h1>Conteúdo da Página.</h1>
                        </body>
                    </html>

Links (Âncoras)

Links são criados com a tag <a>:

  • Abrir Google em nova aba
  • Enviar email
  • Ligar para um número
  • Ir para o rodapé da página
                <!-- Link para um site externo que abre em nova aba -->
                    <a href="https://www.google.com" target="_blank">Abrir Google em nova aba</a>

Mídia

Imagens <img>

A tag <img> exibe uma imagem. O atributo alt é crucial para acessibilidade,
caso a imagem não seja carregada.

                    <img src="./img/imagem.jpg" alt="Plano de fundo com código HTML e CSS de exemplo.">

Figura e Legenda <figure>e <figcaption>

<figure>
    <img src="imagem.jpg" alt="Descrição">
    <figcaption>Alberobello é uma comuna italiana da região da Puglia,

    província de Bari, com cerca de 10.855 habitantes. Estende-se por uma área de 40 km²,

    tendo uma densidade populacional de 271 hab/km². Faz fronteira com Castellana Grotte,

    Fasano, Locorotondo, Martina Franca, Monopoli, Mottola, Noci..</figcaption>
</figure>

Conteúdo Embutido <iframe>

Use <iframe> para incorporar conteúdo de outros sites, como vídeos do YouTube.

Tabelas <table>

Tabelas são ótimas para exibir dados tabulares.

<table>
    <thead> <!-- Cabeçalho -->
        <tr> <!-- Linha -->
            <th>Coluna 1</th> <!-- Célula de cabeçalho -->
            <th>Coluna 2</th>
        </tr>
    </thead>
    <tbody> <!-- Corpo -->
        <tr>
            <td>Dado 1</td> <!-- Célula de dado -->
            <td>Dado 2</td>
        </tr>
    </tbody>
</table>

Formulários <form>

Formulários coletam dados do usuário. A tag <label> é importante para acessibilidade.

<form action="/enviar" method="post">
    <div>
        <label for="nome">Nome:</label>
        <input type="text" id="nome" name="usuario_nome">
    </div>
    <div>
        <label for="email">Email:</label>
        <input type="email" id="email" name="usuario_email">
    </div>
    <button type="submit">Enviar</button>
</form>

Tags Semânticas de Layout

Use estas tags para dar mais significado à estrutura da sua página.

  • <header>: O cabeçalho da página ou de uma seção.
  • <main>: O conteúdo principal e único da página.
  • <section>: Uma seção temática dentro do conteúdo.
  • <article>: Um bloco de conteúdo autônomo (ex: post de blog).
  • <aside>: Conteúdo lateral, como uma barra de links.
  • <footer>: O rodapé da página ou de uma seção.
  • <nav>: Um container para links de navegação.
<body>
    <header>...</header>
    <nav>...</nav>
    <main>
        <section>...</section>
    </main>
    <footer>...</footer>
</body>
